BookNest is a full-stack online bookstore web application developed using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**. The application provides separate modules for **Users, Sellers, and Admins**, allowing users to browse and purchase books, sellers to manage their inventory, and administrators to oversee the entire platform.
